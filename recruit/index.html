Introduction

InfoPlat 希望藉由建立完整的開發流程，形塑設計師及工程師的社群，以及推動校園服務 API 的開放，進而促進開發環境的友善；透過蒐集學生對於校園資訊服務的聲音，讓非開發者也能享受到更為便利的校園。

我們將從臺灣大學開始，推廣資訊開發的風氣，進而將影響擴及到臺灣各大專院校。InfoPlat 的核心價值，就是透過營造 hacker-friendly 的開發環境，讓大學生沒有完成不了的專案。

你想要的校園資訊，讓我們一起開發、一起推廣！
你想要的校園面貌，讓你親身參與、自己打造！
