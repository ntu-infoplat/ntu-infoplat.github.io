Introduction

InfoPlat 希望藉由建立完整的開發流程，形塑設計師及工程師的社群，以及推動校園服務 API 的開放，進而促進開發環境的友善；透過蒐集學生對於校園資訊服務的聲音，讓非開發者也能享受到更為便利的校園。

我們將從臺灣大學開始，推廣資訊開發的風氣，進而將影響擴及到臺灣各大專院校。InfoPlat 的核心價值，就是透過營造 hacker-friendly 的開發環境，讓大學生沒有完成不了的專案。

你想要的校園資訊，讓我們一起開發、一起推廣！
你想要的校園面貌，讓你親身參與、自己打造！

看看 InfoPlat 的合作對象

資訊開發＼臺大開源社

開源社學期間的社課將透過程式專案實作與讀書會的形式，培養社員實際開發的能力，提供開發者一個共同討論、學習技術的環境，同時促進大學生對於開放校園、資訊學權的關注。

使用者經驗＼臺大不一樣思考社 NTU d.thinking

DT將帶領學生秘書的各專案進行design thinking process，透過Empathy、Define、Ideate、Prototype、Test的過程，對於目標使用者的需求進行探討分析；同時擔任各專案的PM，和設計師與工程師溝通協調。

校園媒體＼臺大學生電視台 NTUTBC

TBC已與InfoPlat的「社團平台」專案合作一學期有餘，提供人力拍攝並剪輯各社團的訪問影片，協助資料蒐集；臺大學生報亦於上學期協助社團訪談，擔任記者與編輯的工作。

廣告行銷＼臺大數位影片創作社 NTUDMCC

DMCC將與TBC一同進行社團平台訪問影片的拍攝，替各社團以及InfoPlat的各專案進行形象片的製作，作為行銷與宣傳的利器。
